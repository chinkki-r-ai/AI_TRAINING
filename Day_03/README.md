Personal Expense Tracker
Project Overview

The Personal Expense Tracker is a simple terminal-based Python application that helps users record and manage their daily expenses. Users can add expenses, view all recorded expenses, calculate the total amount spent, and exit the application through an interactive menu.

This project was built to practice Python fundamentals and Object-Oriented Programming (OOP) concepts.

Features

* Add a new expense
* View all recorded expenses
* Calculate the total expense
* Interactive terminal menu
* Simple and beginner-friendly interface

Technologies Used

* Python 3
* Object-Oriented Programming (OOP)

OOP Concepts Used

* Classes
* Objects
* Constructors (`__init__`)
* Methods
